# tg-flood-sender

Инструмент для отправки send_code запросов в Telegram на свои номера телефонов.

## Файлы

- `flood_multi.py` — основной скрипт
- `phones.txt` — список своих номеров (каждый с новой строки)
- `proxies.txt` — список прокси (каждый с новой строки)

## Установка зависимостей

```bash
pip install pyrogram==2.0.106 TgCrypto==1.2.5
```

## Запуск

```bash
python flood_multi.py
```

## Логика работы

- Все номера из phones.txt обрабатываются параллельно
- Прокси из proxies.txt используются циклически (поддерживается rotating proxy)
- При FloodWait — скрипт ждёт, за 120 секунд до конца начинает проверять запросами
- В последние 5 секунд флуда — спамит без паузы чтобы не упустить момент
- Цель: довести флуд на номере до 24 часов

## Формат phones.txt

```
+79991234567
+79997654321
```

## Формат proxies.txt

```
socks5://user:password@host:port
http://user:password@host:port
```
